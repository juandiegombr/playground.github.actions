# CLI Tool

A command-line interface tool built with Node.js and Commander.js.

## Installation

```bash
npm install
```

## Usage

After installation, you can use the following commands:

```bash
# Say hello to the world
cli-tool hello

# Say hello to a specific person
cli-tool hello John
```

## Development

To run tests:

```bash
npm test
```

To run tests in watch mode:

```bash
npm run test:watch
```
